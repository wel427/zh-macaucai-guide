# zh-macaucai-guide

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涉及特定主题的信息整理与展示，不针对任何单个域名或具体网站。

## 项目简介

本仓库旨在以静态 HTML 页面的形式，长期保存并公开一组与特定主题相关的资料。每个页面均为独立文件，可直接通过浏览器访问。仓库本身不提供动态服务，也不包含后端逻辑。

## 目录说明

仓库根目录下存放所有 HTML 页面文件。文件命名遵循简洁、可读的原则，便于直接引用和分享。

- `index.html` — 项目入口页面，提供导航或概览。
- 其余 `.html` 文件 — 各个独立的信息页面，内容互不依赖。

## 页面归档说明

所有页面均为手工整理或从公开来源汇总的静态 HTML 文件。归档原则如下：

- 页面内容保持原始格式，不进行功能性修改。
- 文件更新时，旧版本会保留在 git 历史中，不主动删除。
- 页面内不包含跟踪代码、广告或外部资源（除非必要且注明来源）。

## 维护说明

本项目由维护者不定期更新。更新内容可能包括：

- 新增页面
- 修正已有页面中的错误或过时信息
- 调整页面样式以提升可读性

欢迎通过 Issues 提交问题或建议，但请注意：

- 不接受针对任何具体网站或个人的投诉、举报类内容。
- 不接受要求添加特定域名或链接的请求。
- 维护者保留拒绝或关闭任何 Issue 的权利。

## 使用方式

你可以直接通过 GitHub Pages 访问本仓库的已发布页面，或将整个仓库克隆到本地，在浏览器中打开任意 HTML 文件查看。

## 许可

本仓库内容仅用于信息归档与展示。如需引用或转载，请注明出处。
